# Harutyunyan conjecture

Harutyunyan conjecture states that for any positive integer, the sequence generated by repeating the following rules

$$
x = 
\begin{cases} 
x / 6 & \text{if } x \equiv 0 \pmod{6} \\
x / 4 & \text{if } x \equiv 0 \pmod{4} \\
x / 2 & \text{if } x \equiv 0 \pmod{2} \\
7x + 1 & \text{otherwise}
\end{cases}
$$

will always reach the number 1.
